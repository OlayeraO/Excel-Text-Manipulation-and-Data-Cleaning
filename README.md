Excel Text Manipulation & Data Cleaning Projects
A collection of Microsoft Excel projects focused on text manipulation, data transformation, and data cleaning, demonstrating practical techniques used to prepare raw data for analysis and reporting.
Overview
Across these projects, work was done on combining text from multiple cells, splitting text into structured columns, cleaning inconsistent datasets, replacing and standardizing text characters, and applying multiple text functions to solve complex data manipulation tasks.
