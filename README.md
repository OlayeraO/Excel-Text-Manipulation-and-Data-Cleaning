This repository contains a collection of Microsoft Excel projects focused on text manipulation, data transformation, and data cleaning, demonstrating practical techniques used to prepare raw data for analysis and reporting. Across these projects, I worked on combining text from multiple cells, splitting text into structured columns, cleaning inconsistent datasets, replacing and standardizing text characters, and applying multiple text functions to solve complex data manipulation tasks. The projects were completed using Microsoft Excel and incorporate a wide range of built-in features and functions, including Flash Fill, Text to Columns, Find & Replace, CONCAT, CONCATENATE, TEXTJOIN, LEFT, RIGHT, MID, LEN, FIND, SEARCH, SUBSTITUTE, REPLACE, TRIM, CLEAN, UPPER, LOWER, PROPER, and EXACT. Through these exercises, I developed strong proficiency in formula writing, nested functions, text extraction, string manipulation, spreadsheet organization, data standardization, and data preprocessing, showcasing the practical Excel skills required to transform messy, unstructured data into clean, accurate, and analysis-ready datasets for business intelligence, reporting, and decision-making.
